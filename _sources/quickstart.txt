================

動作環境
------------------------------
以下のものが動作する環境を対象とします。

 * Ruby 1.8/1.9
 * Sqlite3

必要なライブラリのインストール
------------------------------
gemをインストールします。::

 $ sudo gem install oauth
 $ sudo gem install sqlite3-ruby

Ruby1.8の場合は、さらにjsonが必要です。::

 $ sudo gem install json

単体テストを実行するには、さらにrakeとrspecが必要です。::

 $ sudo gem install rspec
 $ sudo gem install rake
